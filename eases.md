# AnimJS easing functions

could be written in different ways, all of them are valid

---

## Linear 
```js
ease: 'linear'

// or

ease: 'none'
```
```js
import { Easing } from '@enea-entertainment/animjs';

ease: Easing.linear

// or

ease: Easing.none
```

---
